Hoja de Diagnóstico del Técnico – PDF Rellenable

Haz clic en el botón para descargar un PDF con campos editables (AcroForm). Abre el PDF en la tablet (por ejemplo, con Adobe Acrobat) para rellenarlo.

¿Cómo lo subo a GitHub Pages?
  1. Crea un repositorio nuevo y sube estos archivos.
  2. En Settings → Pages, elige la rama main y carpeta root.
  3. Abre la URL que te da GitHub Pages y prueba el botón.
